# ID 选择器

在一个HTML文档中,CSS ID 选择器会根据该元素的 ID 属性中的内容匹配元素,元素 ID 属性名必须与选择器中的 ID 属性名完全匹配，此条样式声明才会生效。

/* The element with id="demo" */
#demo {
  border: red 2px solid;
}

# 语法

#id属性值 {样式声明 }
1

下面的属性选择器语句等价：

[id=id属性值] {样式声明 }
1

# 示例

CSS

span#identified {
  background-color: DodgerBlue;
}
