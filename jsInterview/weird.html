                                                                                                                                                                                                                # true + false

                                                                                                                                                                                                                • "truefalse"

                                                                                                                                                                                                                • 1

                                                                                                                                                                                                                • NaN

                                                                                                                                                                                                                • SyntaxError

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                1

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                首先将表达式进行转换-js隐式转换

                                                                                                                                                                                                                Number(true); // -> 1
Number(false); // -> 0
1 + 0; // -> 1

                                                                                                                                                                                                                # [,,,].length

                                                                                                                                                                                                                • 0

                                                                                                                                                                                                                • 3

                                                                                                                                                                                                                • 4

                                                                                                                                                                                                                • SyntaxError

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                3

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                将数组转为string,然后进行length当对后一个元素为空则会省略

                                                                                                                                                                                                                [,] + [,]; // -> ""
[] + [] === [,] + [,]; // -> true
[,,,] + [,,,]; // -> ",,,,"
([,,,] + [,,,]).length === [,,,,].length; // -> true

                                                                                                                                                                                                                # [1, 2, 3] + [4, 5, 6]

                                                                                                                                                                                                                • "123456"

                                                                                                                                                                                                                • "1,2,34,5,6"

                                                                                                                                                                                                                • "1,2,3,4,5,6"

                                                                                                                                                                                                                • NaN

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                “1,2,34,5,6”

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                极其简单的答案是将数组转换为字符串，然后进行连接

                                                                                                                                                                                                                顺便说一下，添加尾随逗号不会改变任何东西：

                                                                                                                                                                                                                [1, 2, 3,] + [4, 5, 6]; // -> "1,2,34,5,6"

                                                                                                                                                                                                                但是，我想，如果您真的想将数组转换为逗号分隔的字符串并将它们组合起来，您可以编写如下愚蠢的代码：

                                                                                                                                                                                                                [1, 2, 3] + [, 4, 5, 6]; // -> "1,2,3,4,5,6"

                                                                                                                                                                                                                或者，更愚蠢的是，这个：

                                                                                                                                                                                                                [1, 2, 3, ""] + [4, 5, 6]; // -> "1,2,3,4,5,6"

                                                                                                                                                                                                                不过，最好不要将加法运算符与数组一起使用。如果您确实想真正组合两个数组，这是一个更好的方法：

                                                                                                                                                                                                                [...[1, 2, 3], ...[4, 5, 6]];

                                                                                                                                                                                                                # 0.2 + 0.1 === 0.3

                                                                                                                                                                                                                • true

                                                                                                                                                                                                                • false

                                                                                                                                                                                                                • NaN

                                                                                                                                                                                                                • SyntaxError

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                false

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                小数的二进制算法除不尽 与其直接比较两个浮点数，不如将浮点数与某种程度的容差进行比较 这个 StackOverflow 答案open in new window

                                                                                                                                                                                                                0.2 + 0.1; // -> 0.30000000000000004;
0.2 + 0.1 > 0.3; // -> true

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                该逗号操作符open in new window只是返回的最后操作数的值。

                                                                                                                                                                                                                10, 2; // -> 2
1, 2, 3, 4; // -> 4
42, "pineapple", true; // -> true

                                                                                                                                                                                                                # !!""

                                                                                                                                                                                                                • true

                                                                                                                                                                                                                • false

                                                                                                                                                                                                                • undefined

                                                                                                                                                                                                                • SyntaxError

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                false

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                可以在任何值之前添加两个感叹号以获取其布尔表示。通常，任何有值的都是真，任何有“空”值的都是假。

                                                                                                                                                                                                                Boolean(""); // -> false
Boolean(0); // -> false
Boolean("Pineapple"); // -> true
Boolean(42); // -> true

                                                                                                                                                                                                                # +!![]

                                                                                                                                                                                                                • true

                                                                                                                                                                                                                • false

                                                                                                                                                                                                                • 0

                                                                                                                                                                                                                • 1

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                1

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                空值通常由布尔值 false 表示。但是，空数组是一个例外。它由 true 表示。然后加号字符将 true 转换为其数字表示。

                                                                                                                                                                                                                Boolean([]); // -> true
Number(true); // -> 1

                                                                                                                                                                                                                • true

                                                                                                                                                                                                                • false

                                                                                                                                                                                                                • 0

                                                                                                                                                                                                                • SyntaxError

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                false

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                !的数量本身没有意义，主要是单数去反，双数仅取当前值得boolean值

                                                                                                                                                                                                                !!!!!!!!!!!!true; // -> true

                                                                                                                                                                                                                # true == "true"

                                                                                                                                                                                                                • true

                                                                                                                                                                                                                • false

                                                                                                                                                                                                                • undefined

                                                                                                                                                                                                                • SyntaxError

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                false

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                根据抽象相等比较open in new window的规则，这两个值都转换为数字。

                                                                                                                                                                                                                Number(true); // -> 1
Number("true"); // -> NaN
1 == NaN; // -> false

                                                                                                                                                                                                                # 010 - 03

                                                                                                                                                                                                                • 7

                                                                                                                                                                                                                • 5

                                                                                                                                                                                                                • 3

                                                                                                                                                                                                                • NaN

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                5

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                010 被 JavaScript 视为八进制数。因此，它的值以 8 为基数。请参阅 Mozilla 对八进制数字的解释open in new window。

                                                                                                                                                                                                                010; // -> 8
03; // -> 3
8 - 3; // -> 5

                                                                                                                                                                                                                如果您愿意，您可以全力以赴使用八进制数：

                                                                                                                                                                                                                01111111111111111; // -> 40210710958665

                                                                                                                                                                                                                顺便说一下，前导零的数量无关紧要：

                                                                                                                                                                                                                010 === 0000000010; // -> true

                                                                                                                                                                                                                # "" - - ""

                                                                                                                                                                                                                • ""

                                                                                                                                                                                                                • 0

                                                                                                                                                                                                                • NaN

                                                                                                                                                                                                                • SyntaxError

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                0

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                这两个空字符串都转换为 0。

                                                                                                                                                                                                                Number(""); // -> 0
0 - - 0; // -> 0

                                                                                                                                                                                                                如果我这样写，表达式可能会变得更清晰一些：

                                                                                                                                                                                                                +"" - -"";
+0 - -0;

                                                                                                                                                                                                                请注意，虽然我在减号和空字符串之间放置空格只是为了让您感到困惑，但减号之间的空格本身很重要：

                                                                                                                                                                                                                - -""; // -> 0
--""; // -> SyntaxError

                                                                                                                                                                                                                # null + 0

                                                                                                                                                                                                                • 0

                                                                                                                                                                                                                • "null0"

                                                                                                                                                                                                                • NaN

                                                                                                                                                                                                                • TypeError

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                0

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                Null 转换为其数字表示形式：0。

                                                                                                                                                                                                                Number(null); // -> 0
0 + 0; // -> 0

                                                                                                                                                                                                                这也意味着，虽然...

                                                                                                                                                                                                                null === false; // -> false

                                                                                                                                                                                                                ... 这是真的：

                                                                                                                                                                                                                +null === +false; // -> true

                                                                                                                                                                                                                # 0/0

                                                                                                                                                                                                                • 0

                                                                                                                                                                                                                • Infinity

                                                                                                                                                                                                                • NaN

                                                                                                                                                                                                                • SyntaxError

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                NaN

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                0/0 没有有意义的数字答案，因此输出只是NaN。

                                                                                                                                                                                                                isNaN(0/0); // -> true

                                                                                                                                                                                                                # 1/0 > Math.pow(10, 1000)

                                                                                                                                                                                                                • true

                                                                                                                                                                                                                • false

                                                                                                                                                                                                                • NaN

                                                                                                                                                                                                                • SyntaxError

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                false

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                JavaScript 将这两个值都视为无穷大，无穷大等于无穷大了解有关无穷大的更多信息open in new window

                                                                                                                                                                                                                1/0; // -> Infinity
Math.pow(10, 1000); // -> Infinity
Infinity > Infinity; // -> false

                                                                                                                                                                                                                # true++

                                                                                                                                                                                                                • 2

                                                                                                                                                                                                                • undefined

                                                                                                                                                                                                                • NaN

                                                                                                                                                                                                                • SyntaxError

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                SyntaxError

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                undefined++不会导致 SyntaxError：

                                                                                                                                                                                                                1++; // -> SyntaxError
"x"++; // -> SyntaxError
undefined++; // -> NaN

                                                                                                                                                                                                                当然，为了完全清楚，这是有效的语法：

                                                                                                                                                                                                                let _true = true;
_true++;
_true; // -> 2

                                                                                                                                                                                                                # "" - 1

                                                                                                                                                                                                                • "1"

                                                                                                                                                                                                                • "-1"

                                                                                                                                                                                                                • -1

                                                                                                                                                                                                                • NaN

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                -1

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                加法运算符 (+) 用于数字和字符串，减法运算符 (-) 不用于字符串，因此 JavaScript 将其解释为数字之间的运算。空字符串转换为 0。

                                                                                                                                                                                                                Number(""); // -> 0
0 - 1; // -> -1;

                                                                                                                                                                                                                即使字符串内部有空格（或更多），这仍然是正确的：

                                                                                                                                                                                                                " " - 1; // -> -1;

                                                                                                                                                                                                                但是，如果使用加法运算符，则字符串连接优先：

                                                                                                                                                                                                                "" + 1; // -> "1";

                                                                                                                                                                                                                # (null - 0) + "0"

                                                                                                                                                                                                                • "null0"

                                                                                                                                                                                                                • "00"

                                                                                                                                                                                                                • 0

                                                                                                                                                                                                                • NaN

                                                                                                                                                                                                                # 正确答案

                                                                                                                                                                                                                "00"

                                                                                                                                                                                                                # 思路

                                                                                                                                                                                                                Number(null) - 0; // -> 0
0 + "0"; // -> "00"

                                                                                                                                                                                                                如果问题只使用了减法运算符，结果就会不同：

                                                                                                                                                                                                                (null - 0) - "0"; // -> 0

                                                                                                                                                                                                                # true + ("true" - 0)

                                                                                                                                                                                                                • 1

                                                                                                                                                                                                                [].toString(); // -> ""

                                                                                                                                                                                                                由于尾随逗号，这些表达式是相等的：

                                                                                                                                                                                                                [] + [] === [,] + [,]; // -> true

                                                                                                                                                                                                                当然，这也是事实：

                                                                                                                                                                                                                Number([]) === Number([,]); // -> true
